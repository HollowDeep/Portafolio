Introduccion a Eventos

Los eventos son muy importantes para lograr que las paginas sean interactivas para los usuarios. Podemos controlar los eventos con movimientos de raton, clicks o precionando alguna tecla.

El objeto `Event` es una representacion de un evento que ocurre en el navegador. Cada vez que un usuario interactua con la pagina, se genera un evento que puede ser capturado y manejado, a eso se le llama event handlers o tambien listener de eventos

Conceptos importantes.

Event flow.

Esta wea es flujo de eventos. Cuando hay multiples eventos, estos tendran un flujo con el cual se van a ejecutar. Estos flujos pueden ser Event bubbling o Event Capturing.

Event bubbling.

Su pongamos que tenemos una jerarquia de div.padre>div.hijo>button y el button tiene un evento al igual que los div. Cuando se maneja este tipo de flujo, primero se ejecutara el evento button, luego el div-hijo, despues el div-padre. Podemos entender que es un suceso en cadena, o que burbujea y detona los demas eventos.

Event Capturing.

Usando la jerarquia anterior, este tipo de flujo funciona al revez, primero se ejecuta el div-padre, luego el div-hijo y por ultimo el button

Tipos de eventos

Los tipos de eventos que pueden ser manejados por JS son:

  • Eventos de mouse: click, dbclick, mouseover, mousemove, etc.
  • Eventos de teclado: keydown, keyup, keypress, etc.
  • Eventos de formulario: submit, change, focus, blur
  • Eventos de ventana: load, resize, scroll
